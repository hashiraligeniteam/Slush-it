




            ******* KILLER SPLAT *******
            
            
            
            
          - HOW TO RUN PROJECT IN UNITY -
          
1. Open the scene "Gameplay"
2. Click on play and enjoy the game




               - HOW TO PLAY GAME -
              
1. First drag the cube on the top side of any dummy (Red capsule)
2. Release the click, it will drop down on dummy
3. Then use swipe gestures (Right, left, up and down) to fill in the black tiles with red blood
4. Kill other dummies to get more blood
5. Fill all the tiles to complete the level
